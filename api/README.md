#项目介绍
##模块介绍
###wms-project-seed
    项目基础种子模块，项目其他模块都要引入模块

###wms-api
    后台服务统一对外开放的接口服务，供前端页面、app端、微信、小程序调用
 
###wms-system-manage
    系统管理模块，提供账号管理、角色管理、角色权限设置、机构管理、系统参数管理、个人信息维护、注册、登录、修改密码、退出系统等功能

###wms-intercept
    系统拦截模块，拦截访问后台接口的请求，根据token判断请求是否有访问目标api接口权限

 
    运营管理模块

###wms-message
    消息模块，包括咨询、消息待办、联系客服、在线帮助等 

###wms-file
    系统文件管理模块，包括文件上传、显示等
